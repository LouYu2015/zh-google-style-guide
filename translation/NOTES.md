# 翻译注意事项与 Prompt 记录

本文件记录翻译过程中的决策、AI Prompt 模板和进度追踪。

术语对照表见 [GLOSSARY.md](GLOSSARY.md)。

---

## 翻译原则

1. **忠实原文**：保持与原文语义的一致性，不擅自增减内容
2. **技术术语**：首次出现时保留英文原文，格式为 `中文（English）`
3. **代码示例**：代码块内容保持原文不变，仅翻译注释
4. **语气风格**：使用正式、简洁的中文技术文档风格

---

## AI 翻译 Prompt 模板

### 通用翻译 Prompt

```
你是一位专业的技术文档翻译专家，请将以下 Google Style Guide 的章节从英文翻译为简体中文。

翻译要求：
1. 保持技术术语的准确性，首次出现的专业术语保留英文原文，格式：中文（English）
2. 代码块内容保持原文不变，仅翻译代码注释（// 或 # 后的注释）
3. 使用正式、简洁的中文技术文档风格，避免口语化表达
4. 保持原文的 Markdown 格式结构不变
5. 参照 translation/GLOSSARY.md 中的术语对照表保持术语一致性

请翻译以下内容：
[在此粘贴原文]
```

### 审校 Prompt

```
请审校以下 Google Style Guide 中文翻译，重点检查：
1. 技术术语是否准确且与术语表（translation/GLOSSARY.md）一致
2. 语言是否流畅自然，符合中文技术文档习惯
3. 是否有遗漏或误译的段落
4. Markdown 格式是否完整保留

原文：
[原文]

翻译：
[翻译]
```

### 术语提取 Prompt

```
请阅读以下 Google Style Guide 章节，提取所有重要的技术术语，
并为每个术语提供建议的中文译法和简短说明。
输出格式为 Markdown 表格：| 英文 | 建议中文 | 说明 |

[在此粘贴原文]
```
