# 术语对照表

翻译时应保持以下术语的统一译法。首次出现的专业术语在正文中保留英文原文，格式为 `中文（English）`。

| 英文 | 中文 | 说明 |
|------|------|------|
| Style Guide | 风格指南 | 不译为"代码规范" |
| Convention | 约定 | 优先于"惯例" |
| Best Practice | 最佳实践 | 通用译法 |
| Linter | Linter | 保留英文 |
| Namespace | 命名空间 | C++ 专用 |
| Scope | 作用域 | 通用 |
| Header file | 头文件 | C++ 专用 |
| Dependency | 依赖 | 优先于"依存" |
| Deprecated | 已废弃 | 优先于"过时" |
| Template | 模板 | C++ 上下文 |
| Refactoring | 重构 | 通用 |
| Boilerplate | 样板代码 | 通用 |
| Callback | 回调 | 保留含义 |
| Interface | 接口 | 通用 |
| Abstract | 抽象 | 通用 |
