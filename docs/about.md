# 关于本项目

## 项目简介

本项目旨在为中文开发者提供 [Google Style Guide](https://github.com/google/styleguide)
的高质量中文翻译，采用 AI 辅助翻译与人工审校相结合的方式。

## 翻译方法

1. **AI 初译**：使用大语言模型（LLM）对原文进行初步翻译
2. **术语统一**：参照 `translation/GLOSSARY.md` 中的术语表保持一致性
3. **人工审校**：对技术细节和语言表达进行人工复核
4. **版本跟踪**：通过 git submodule 跟踪原文更新

## 贡献

欢迎提交 Issue 或 Pull Request，尤其是：

- 翻译错误修正
- 术语建议
- 格式改进
